




# StudyTracker 🎓

**StudyTracker** — это полноценное веб-приложение на Flask, разработанное для абитуриентов и школьников. Оно помогает структурировать подготовку к экзаменам, управлять задачами по разным предметам и отслеживать прогресс для поступления в ВУЗ мечты.

## ✨ Основные функции

- **Управление предметами:** создание учебных дисциплин с установкой целевого балла и цветовой маркировкой.
- **Трекер задач:** система управления задачами с дедлайнами, приоритетами (Low, Medium, High) и статусами выполнения.
- **Дашборд:** наглядная статистика общего прогресса подготовки и список ближайших задач.
- **Хранение материалов:** возможность загружать учебные файлы (PDF, изображения) напрямую к задачам.
- **Личный профиль:** настройка целевого ВУЗа (МИФИ, СГТУ, СГУ и др.) и загрузка аватара.
- **Антистресс-модуль:** интеграция с внешним API для получения рандомных фото собак для психологической разгрузки.

## 🛠 Технологический стек

* **Backend:** Python 3.x, Flask
* **Database:** SQLite + SQLAlchemy (ORM)
* **Auth:** Flask-Login, Flask-Bcrypt
* **Forms:** Flask-WTF + WTForms
* **Frontend:** Jinja2, Bootstrap 5, HTML/CSS

## 🚀 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/pou220/StudyTracker.git
   cd StudyTracker
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для Linux/macOS:
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите приложение:**
   ```bash
   python app.py
   ```
   После этого приложение будет доступно по адресу `http://127.0.0.1:5000/`.

## 📂 Структура проекта

- `app.py` — точка входа и конфигурация приложения.
- `models.py` — описание моделей базы данных (User, Subject, Task, TaskFile).
- `routes.py` — логика обработки маршрутов и контроллеры.
- `forms.py` — формы регистрации, авторизации и редактирования данных.
- `static/` — CSS, изображения и загруженные пользователями файлы.
- `templates/` — HTML-шаблоны страниц.

## 🛡 Безопасность

- Хеширование паролей с помощью `bcrypt`.
- Защита от CSRF-атак во всех формах.
- Проверка прав доступа: пользователь может видеть и редактировать только свои данные.
- Валидация типов и размеров загружаемых файлов.

Тема проекта: Разработка веб-приложения для отслеживания прогресса подготовки к экзаменам «StudyTracker».



Цель проекта: Создать многопользовательский веб-сервис, помогающий абитуриентам структурировать процесс подготовки к экзаменам, устанавливать целевые баллы, управлять задачами и отслеживать личную эффективность.



Основные задачи:

1. Спроектировать и реализовать реляционную базу данных для хранения информации о пользователях, учебных предметах, задачах и прикрепленных файлах.
2. Разработать систему регистрации и авторизации пользователей с безопасным хешированием паролей.
3. Создать интерфейс для управления учебными предметами (CRUD-операции, настройка целевых баллов и цветовой маркировки).
4. Реализовать функционал управления задачами (дедлайны, приоритеты, статусы выполнения).
5. Настроить безопасную загрузку и хранение пользовательских файлов (аватары, материалы к задачам).
6. Разработать информативный дашборд с выводом статистики успеваемости и ближайших дедлайнов.
7. Интегрировать стороннее API для реализации модуля «Антистресс».



Требования к технологическому стеку:

* Backend: Python 3, фреймворк Flask.
* СУБД: SQLite (через ORM SQLAlchemy).
* Frontend: HTML5, шаблонизатор Jinja2, CSS (Bootstrap).
* Безопасность: Flask-Login (управление сессиями), Flask-Bcrypt (хеширование), Flask-WTF (CSRF-защита и валидация форм).



Ожидаемый результат: Готовое к развертыванию веб-приложение с интуитивно понятным интерфейсом, полностью покрывающее заявленный функционал.
